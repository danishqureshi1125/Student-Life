About Student Life

Helping students stay organized, productive, and balanced.

Purpose of the Website

Student Life is designed to give college students a simple place to find advice about studying, productivity, mental health, and everyday responsibilities. It focuses on practical tips that are easy to understand and apply.

Who This Website Helps

This platform is built for college students who want to improve their habits, reduce stress, and become more productive. It supports students managing exams, deadlines, work, and personal life.

Project Goal

This project demonstrates how a website can be built and deployed using cloud technology. It was developed using HTML, CSS, and JavaScript, managed through GitHub, and hosted using AWS.
