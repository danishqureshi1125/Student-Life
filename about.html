About Student Life

Helping students stay organized, motivated, and balanced.

Purpose of the Website

Student Life is a website created to help college students manage academics, productivity, mental health, and everyday responsibilities. The goal is to give students practical tips that are simple, useful, and easy to apply.

Who This Website Helps

This website is designed for college students who want to improve their study habits, reduce stress, manage deadlines, and build healthier routines during the semester.

Project Goal

This project was created for ITSS 4371 to demonstrate how a website can be built and deployed using cloud technology. The website was developed with HTML, CSS, and JavaScript, managed through GitHub, and hosted using AWS.
